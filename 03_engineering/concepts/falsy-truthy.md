# Falsy vs Truthy

## What it is
Here are the Falsy 8 : 
- `false` / Boolean
- `0` / Number
- `-0` / Number
- `0n` / BigInt (integer values too big or too small to be represented by the number primitive)
- `' '` or `" "` / String
- `null` / Object 
- `undefined` / Undefined
- `NaN` / Number

BUT, there are edge cases :
- `[]` and `{}` are TRUTHY because in JS, objects and arrays are truthy even if empty
- `"false"` and `"0"` are TRUTHY because they are strings that are not technically empty.
